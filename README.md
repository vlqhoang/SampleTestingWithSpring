# SampleTestingWithSpring
This is a simple project applying Junit5 Automation Test demonstrating how JunitTests work.

## Components
* Junit5 Unit Tests.
* Junit5 Integration Tests.

## Techs
* **Junit5 features.**
* **Spring Boot, Spring Data JPA.**
* **H2 Database.**

## Usage
#### Maven
```bash
mvn clean install
```
